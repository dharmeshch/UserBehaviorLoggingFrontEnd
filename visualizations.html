Logged Actions

In this assignment 5 different actions of the user on the website are logged. They are

  • Average time spent by the user in reading the question
  • Number of hits on each question
  • Likes and Dislikes
  • Bookmarks
  • New Posts

The amount of time spent on each question will help us understand the usefulness of the question. All the other statistics of the user like the number of questions posted, will help us categorise the users according to their activity. The number of questions bookmarked and the number of questions liked/disliked will specify the usefulness of the question and the validity of the question. The number of hits for a question is logged to analyse the topics the user are interested in. All these actions can help in personalizing the website specific to user.
